## 说明
据不可靠信息，NOJ会对代码进行筛查，请**不要**直接复制代码，至少改变变量名等，当然参照标准代码修改自己程序我们更为倡导

本项目助力每一位在调程序的nwpuer，提供一份标准代码用于程序调试，或在期末关键时刻提供一份代码用于补救

希望每一位都能体验到Coding的魅力
